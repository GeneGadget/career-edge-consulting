## How to Set Up Add-On Event Types

Since you have add-ons that require booking after the basic package, here's how to set up additional event types in Calendly:

### Step 1: Create Additional Event Types in Calendly

1. **Log into Calendly** â†’ Go to [calendly.com](https://calendly.com) and sign in
2. **Create New Event Type:**
   - Click **"Event Types"** in the left sidebar
   - Click **"+ New Event Type"** or **"Create"**
   - Choose **"One-on-One"** (recommended for consulting sessions)

### Step 2: Set Up Each Add-On Event Type

#### Add-On 1: Extra Page / Full Resume Rewrite-Reformat ($25)
1. **Event Type Name:** "Resume Rewrite/Reformat Add-On"
2. **Duration:** 30-45 minutes (or as needed)
3. **Description:** 
   ```
   Add-on service for clients who have completed the basic 2-page resume review.
   
   This includes:
   - Extra page addition OR
   - Full resume rewrite and reformatting
   
   Price: $25 CAD
   Requires: Basic package booking first
   ```
4. **Scheduling Options:**
   - Set availability (Mon-Fri 6-9 PM, Weekends 9 AM-5 PM)
   - Buffer time if needed
5. **Questions/Form Fields:**
   - Add checkbox: "I have completed the basic resume review package"
   - Optional: Ask for their original resume reference
6. **Save** and note the event URL (e.g., `https://calendly.com/careeredgeconsulting/resume-rewrite`)

#### Add-On 2: Job Search and Strategy ($50 - 30 minutes)
1. **Event Type Name:** "Job Search Strategy Session"
2. **Duration:** 30 minutes
3. **Description:**
   ```
   Add-on service for clients who have completed the basic 2-page resume review.
   
   This session covers:
   - LinkedIn profile optimization
   - Networking strategies (Facebook groups, websites, LinkedIn)
   - Hidden job market tactics
   - Personal branding
   
   Price: $50 CAD - 30 minutes
   Requires: Basic package booking first
   ```
4. **Scheduling Options:** Same as above
5. **Questions/Form Fields:**
   - Add checkbox: "I have completed the basic resume review package"
   - Ask about their target industry/roles
6. **Save** and note the event URL (e.g., `https://calendly.com/careeredgeconsulting/job-strategy`)

#### Add-On 3: 30-Minute Coaching ($25 - 30 minutes)
1. **Event Type Name:** "30-Minute Career Coaching"
2. **Duration:** 30 minutes
3. **Description:**
   ```
   Add-on service for clients who have completed the basic 2-page resume review.
   
   Personalized 30-minute coaching session focused on your specific career questions and goals.
   
   Price: $25 CAD - 30 minutes
   Requires: Basic package booking first
   ```
4. **Scheduling Options:** Same as above
5. **Questions/Form Fields:**
   - Add checkbox: "I have completed the basic resume review package"
   - Ask about their main career goals/questions
6. **Save** and note the event URL (e.g., `https://calendly.com/careeredgeconsulting/30min-coaching`)

### Step 3: Configure File Upload (Important!)

For the **Basic Package** event type:
1. Go to your event type â†’ **"Questions"** tab
2. Enable **"File Upload"** or **"Document Upload"**
3. Add a question: "Please upload your current resume (PDF or Word format)"
4. This allows clients to upload their resume during booking

For add-on event types:
- File upload is optional (they may have already provided it in the basic package)

### Step 4: Set Availability

For all event types:
1. Go to **"Availability"** tab in each event type
2. Set your availability:
   - **Monday - Friday:** 6:00 PM - 9:00 PM EST
   - **Weekends:** 9:00 AM - 5:00 PM EST
3. Set timezone to Eastern Time (EST/EDT)

### Step 5: Payment Setup (If Using Calendly Payments)

If you want to accept payment through Calendly:

1. **Connect Payment Method:**
   - Go to **Settings** â†’ **Integrations** â†’ **Payments**
   - Connect Stripe or PayPal

2. **Set Prices for Each Event:**
   - Basic Package: $50 (or $30 for students - you may need a separate event type)
   - Resume Rewrite Add-On: $25
   - Job Search Strategy: $50
   - 30-Minute Coaching: $25

3. **Enable Payment Required:**
   - In each event type â†’ **"Additional Options"** â†’ Enable **"Payment Required"**
   - Set the price for each event type

**Note:** Calendly's basic plan may have limitations on payments. Check your plan features.
